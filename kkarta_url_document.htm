Några query parameters  för att dölja knapparna från Kristianstadskartan:

Kristianstadskartan huvud url:

http://kartor.kristianstad.se/kristianstadskartan/

 

Knapparna för att dölja med hjälp av URL och sök från URL

Query Parameters

Exempel URL

Bild

1.       Möjlighet att via kart-URL:en ange en eller flera adresser för att visa en eller flera markörer

poim= kommaseparerade address eller

&poim= kommaseparerade address

 

 

poim betyder “points of interest multiple”.

En address sök:

https://kartor.kristianstad.se/kristianstadskartan/?&poim=KRISTIANSTAD S:1

 

Flera address sök:

https://kartor.kristianstad.se/kristianstadskartan/?&poim=KRISTIANSTAD S:1,KRISTIANSTAD S:2,KRISTIANSTAD S:3

 

eller

 

https://kartor.kristianstad.se/kristianstadskartan/?&poim=KRISTIANSTAD%20S:1,KRISTIANSTAD%20S:2,KRISTIANSTAD%20S:3

 

 

2.       Möjlighet att via kart-URL:en ange en eller flera positioner för att visa en eller flera markörer

För GPS-koordinater:

xym= kommaseparerade longitude latidute

 

eller

 

&xym= kommaseparerade longitude latidute

 

För SWEREF99 13 30-koordinater:

xym= kommaseparerade y,x koordinater

 

eller

&xym= kommaseparerade y,x koordinater

 

För GPS-koordinater:

https://kartor.kristianstad.se/kristianstadskartan/?xym=14.1529,56.0354,14.1528,56.0351,14.1539,56.0355

 

              

               För SWEREF99 13 30-koordinater:

https://kartor.kristianstad.se/kristianstadskartan/?xym=190698.66651196362,6212696.850608168,190692.74873832482,6212663.389770443,190760.89532623536,6212708.574182627

 

 

3.       Lägg till en parameter så det kan lägga text på info-dialog i ordning för position och adress sök från URL

Texten kommer att följa ordningen i adressen eller positioner.

diainfo=[["Kommunal Förskola, Östra"],["Fristående Grundskola, Östra"]]

eller

&diainfo=[["Kommunal Förskola, Östra"],["Fristående Grundskola, Östra"]]

 

 

För adress:

https://kartor.kristianstad.se/kristianstadskartan/?poim=Kockumsgatan%207,Lasarettsboulevarden%2024&diainfo=[["Kommunal Förskola, Östra"],["Fristående Grundskola, Östra"]]

 

https://kartor.kristianstad.se/kristianstadskartan/?poim=Kockumsgatan%207&diainfo=[[%22Kommunal%20F%C3%B6rskola,%20%C3%96stra%22]]

 

För position:

https://kartor.kristianstad.se/kristianstadskartan/?xym=14.1593,56.0401,14.1745,56.0329&diainfo=[["Kommunal Förskola, Östra"],["Fristående Grundskola, Östra"]]

https://kartor.kristianstad.se/kristianstadskartan/?xym=14.1593,56.0401&diainfo=[[%22Kommunal%20F%C3%B6rskola,%20%C3%96stra%22]]
